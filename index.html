What is the DOM?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.

What are event listeners?

An event listener is a procedure or function in a computer program that waits for an event to occur. In JavaScript, we use `element.addEventListener()` to listen for events like clicks, key presses, or mouse movements on HTML elements.
